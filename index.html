輸入 Markdown

預覽與下載

點擊「轉換預覽」按鈕來查看結果

Word 文件 (.docx) 下載

點擊「下載 Word」按鈕，將 Markdown 轉為 .docx 文件。

轉換說明：

  • 保留標題、段落、列表、粗體/斜體、表格等格式。
  • 圖片需為公開 URL，否則可能無法轉換。
  • 複雜 CSS 樣式可能簡化。
  • 程式碼區塊轉為單一字體文字。

注意：客戶端轉換效果有限，請檢查輸出文件。

PPT 文件 (.pptx) 下載

點擊「下載 PPT」按鈕，將 Markdown 轉為 .pptx 文件。

投影片生成規則：

  • # H1 或 --- 開始新投影片。
  • ## H2 作為內容標題。
  • 段落、列表、表格、程式碼區塊加入內容區。
  • 表格和程式碼區塊格式可能簡化。

注意：依賴 Markdown 結構，請檢查輸出簡報。
